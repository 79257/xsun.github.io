File netbus-brute

脚本使用类型: portrule
脚本所属分类: brute, intrusive
脚本下载地址: http://nmap.org/svn/scripts/netbus-brute.nse

User Summary

Performs brute force password auditing against the Netbus backdoor ("remote administration") service.

Script Arguments

passdb, unpwdb.passlimit, unpwdb.timelimit, unpwdb.userlimit, userdb

See the documentation for the unpwdb library.

Example Usage

执行格式 

nmap -p 12345 --script netbus-brute <target>

Script Output 

12345/tcp open  netbus
|_netbus-brute: password123

Requires

  • nmap
  • shortport
  • stdnse
  • string
  • unpwdb
