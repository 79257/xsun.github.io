File drda-info

脚本使用类型: portrule
脚本所属分类: safe, discovery, version
脚本下载地址: http://nmap.org/svn/scripts/drda-info.nse

User Summary

Attempts to extract information from database servers supporting the DRDA protocol. The script sends a DRDA EXCSAT (exchange server attributes) command packet and parses the response.

Example Usage

执行格式 

nmap -sV <target>

Script Output 

PORT      STATE SERVICE
50000/tcp open  drda
|  drda-info: DB2 Version: 8.02.9
|  Server Platform: QDB2/SUN
|  Instance Name:   db2inst1
|_ External Name:   db2inst1db2agent00002B430
