File bitcoin-info

脚本使用类型: portrule
脚本所属分类: discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/bitcoin-info.nse

User Summary

Extracts version and node information from a Bitcoin server

Example Usage

执行格式 

nmap -p 8333 --script bitcoin-info <ip>

Script Output
