File maxdb-info

脚本使用类型: portrule
脚本所属分类: default, version
脚本下载地址: http://nmap.org/svn/scripts/maxdb-info.nse

User Summary

Retrieves version and database information from a SAP Max DB database.

Example Usage

执行格式 

nmap -p 7210 --script maxdb-info <ip>
