User Summary

Retrieves information from an Apache Hadoop JobTracker HTTP status page.

Information gathered: * State of the JobTracker. * Date/time the service was started * Hadoop version * Hadoop Compile date * JobTracker ID * Log directory (relative to http://host:port/) * Associated TaskTrackers * Optionally also user activity history

For more information about Hadoop, see: * http://hadoop.apache.org/ * http://en.wikipedia.org/wiki/Apache_Hadoop * http://wiki.apache.org/hadoop/JobTracker
