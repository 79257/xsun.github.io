File snmp-processes

脚本使用类型: portrule
脚本所属分类: default, discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/snmp-processes.nse

User Summary

Attempts to enumerate running processes through SNMP.

Script Arguments

snmpcommunity

See the documentation for the snmp library.

Example Usage

执行格式 

nmap -sV -sC <target>
