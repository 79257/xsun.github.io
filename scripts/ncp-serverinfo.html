File ncp-serverinfo

脚本使用类型: portrule
脚本所属分类: default, discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/ncp-serverinfo.nse

User Summary

Retrieves eDirectory server information (OS version, server name, mounts, etc.) from the Novell NetWare Core Protocol (NCP) service.

Example Usage

执行格式 

nmap -sV -sC <target>
