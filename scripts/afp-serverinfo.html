File afp-serverinfo

脚本使用类型: portrule
脚本所属分类: default, discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/afp-serverinfo.nse

User Summary

Shows AFP server information. This information includes the server's hostname, IPv4 and IPv6 addresses, and hardware type (for example Macmini or MacBookPro).

Example Usage

执行格式 

nmap -sV -sC <target>
