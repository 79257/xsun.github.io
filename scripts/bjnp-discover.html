File bjnp-discover

脚本使用类型: portrule
脚本所属分类: safe, discovery
脚本下载地址: http://nmap.org/svn/scripts/bjnp-discover.nse

User Summary

Retrieves printer or scanner information from a remote device supporting the BJNP protocol. The protocol is known to be supported by network based Canon devices.

Example Usage

执行格式 

sudo nmap -sU -p 8611,8612 --script bjnp-discover <ip>
