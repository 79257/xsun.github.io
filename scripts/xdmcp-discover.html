File xdmcp-discover

脚本使用类型: portrule

User Summary

Requests an XDMCP (X display manager control protocol) session and lists supported authentication and authorization mechanisms.

Example Usage

执行格式 

nmap -sU -p 177 --script xdmcp-discover <ip>

Script Output
