User Summary

Discovers information such as log directories from an Apache Hadoop DataNode HTTP status page.

Information gathered: * Log directory (relative to http://host:port/)

For more information about hadoop, see: * http://hadoop.apache.org/ * http://en.wikipedia.org/wiki/Apache_Hadoop * http://wiki.apache.org/hadoop/DataNode

Script Arguments

smbdomain, smbhash, smbnoguest, smbpassword, smbtype, smbusername

See the documentation for the smbauth library.

http.max-cache-size, http.max-pipeline, http.pipeline, http.useragent

See the documentation for the http library.

Example Usage

执行格式 

nmap --script hadoop-datanode-info.nse -p 50075 host

50075/tcp open  hadoop-datanode syn-ack
