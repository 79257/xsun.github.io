File telnet-brute

脚本使用类型: portrule
脚本所属分类: brute, intrusive
脚本下载地址: http://nmap.org/svn/scripts/telnet-brute.nse

User Summary

Tries to get Telnet login credentials by guessing usernames and passwords.

Script Arguments

passdb, unpwdb.passlimit, unpwdb.timelimit, unpwdb.userlimit, userdb

See the documentation for the unpwdb library.

Example Usage

执行格式 

nmap -sV --script=telnet-brute <target>

Script Output 

PORT   STATE SERVICE
23/tcp open  telnet
|_telnet-brute: root - 1234

Requires

  • comm
  • nmap
  • shortport
  • stdnse
  • strbuf
  • string
  • unpwdb
