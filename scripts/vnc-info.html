File vnc-info

脚本使用类型: portrule
脚本所属分类: default, discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/vnc-info.nse

User Summary

Queries a VNC server for its protocol version and supported security types.

Example Usage

执行格式 

nmap -sV -sC <target>

Script Output 

PORT    STATE SERVICE
5900/tcp open  vnc
| vnc-info:  
|   Protocol version: 3.889
|   Security types:
|     Mac OS X security type (30)
|_    Mac OS X security type (35)

Requires

  • shortport
  • stdnse
  • table
  • vnc
