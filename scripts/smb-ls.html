File smb-ls

脚本使用类型: hostrule
脚本所属分类: discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/smb-ls.nse

User Summary

Attempts to retrieve useful information about files shared on SMB volumes. The output is intended to resemble the output of the UNIX ls command.

Script Arguments

randomseed, smbbasic, smbport, smbsign

See the documentation for the smb library.

smbdomain, smbhash, smbnoguest, smbpassword, smbtype, smbusername

See the documentation for the smbauth library.

Example Usage

执行格式 

nmap -sV --script=smb-ls <target>

Requires

  • bit
  • smb
  • stdnse
  • tab
  • table
  • openssl
