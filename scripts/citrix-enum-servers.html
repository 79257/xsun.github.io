File citrix-enum-servers

脚本使用类型: portrule
脚本所属分类: discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/citrix-enum-servers.nse

User Summary

Extracts a list of Citrix servers from the ICA Browser service.

Example Usage

执行格式 

sudo ./nmap -sU --script=citrix-enum-servers -p 1604

Script Output 

PORT     STATE SERVICE
1604/udp open  unknown
| citrix-enum-servers:  
|   CITRIXSRV01  
|_  CITRIXSRV02
