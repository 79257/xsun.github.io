File pptp-version

脚本使用类型: portrule
脚本所属分类: version
脚本下载地址: http://nmap.org/svn/scripts/pptp-version.nse

User Summary

Attempts to extract system information from the point-to-point tunneling protocol (PPTP) service.

Example Usage

执行格式 

nmap -sV <target>

Script Output 

PORT     STATE SERVICE VERSION
1723/tcp open  pptp    YAMAHA Corporation (Firmware: 32838)
Service Info: Host: RT57i

Requires

  • comm
  • nmap
  • shortport
  • string
