File netbus-version

脚本使用类型: portrule
脚本所属分类: version
脚本下载地址: http://nmap.org/svn/scripts/netbus-version.nse

User Summary

Extends version detection to detect NetBuster, a honeypot service that mimes NetBus.

Example Usage

执行格式 

nmap -sV -p 12345 --script netbus-version <target>

Script Output 

12345/tcp open  netbus  Netbuster (honeypot)

Requires

  • nmap
  • shortport
  • stdnse
