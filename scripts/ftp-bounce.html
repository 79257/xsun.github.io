File ftp-bounce

脚本使用类型: portrule
脚本所属分类: default, safe
脚本下载地址: http://nmap.org/svn/scripts/ftp-bounce.nse

User Summary

Checks to see if an FTP server allows port scanning using the FTP bounce method.

Script Arguments

ftp-bounce.password

Password to log in with. Default "IEUser@".

ftp-bounce.username

Username to log in with. Default "anonymous".

Example Usage

执行格式 

nmap -sV -sC <target>

Script Output 

PORT   STATE SERVICE
21/tcp open  ftp
|_ftp-bounce: bounce working!

PORT   STATE SERVICE
21/tcp open  ftp
|_ftp-bounce: server forbids bouncing to low ports <1025

PORT   STATE SERVICE
21/tcp open  ftp
|_ftp-bounce: no banner

Requires

  • coroutine
  • nmap
