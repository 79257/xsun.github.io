File iscsi-info

脚本使用类型: portrule
脚本所属分类: default, safe, discovery
脚本下载地址: http://nmap.org/svn/scripts/iscsi-info.nse

User Summary

Collects and displays information from remote iSCSI targets.

Example Usage

执行格式 

nmap -sV -sC <target>
