File redis-info

脚本使用类型: portrule
脚本所属分类: discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/redis-info.nse

User Summary

Retrieves information (such as version number and architecture) from a Redis key-value store.

Example Usage

执行格式 

nmap -p 6379 <ip> --script redis-info
