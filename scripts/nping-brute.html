File nping-brute

脚本使用类型: portrule
脚本所属分类: brute, intrusive
脚本下载地址: http://nmap.org/svn/scripts/nping-brute.nse

User Summary

Performs brute force password auditing against an Nping Echo service.

See http://nmap.org/book/nping-man-echo-mode.html for Echo Mode documentation.

Script Arguments

passdb, unpwdb.passlimit, unpwdb.timelimit, unpwdb.userlimit, userdb

See the documentation for the unpwdb library.

Example Usage

执行格式 

nmap -p 9929 --script nping-brute <target>

Script Output 

9929/tcp open  nping-echo
| nping-brute: 
|   Accounts
|     123abc => Valid credentials
|   Statistics
|_    Perfomed 204 guesses in 204 seconds, average tps: 1
