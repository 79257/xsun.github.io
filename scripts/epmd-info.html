File epmd-info

脚本使用类型: portrule
脚本所属分类: default, discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/epmd-info.nse

User Summary

Connects to Erlang Port Mapper Daemon (epmd) and retrieves a list of nodes with their respective port numbers.

Example Usage

执行格式 

nmap -p 4369 --script epmd-info <target>

Script Output 

PORT     STATE SERVICE
4369/tcp open  epmd
| epmd-info.nse: 
|   epmd running on port 4369
|   name rabbit at port 36804
|_  name ejabberd at port 46540

Requires

  • bin
  • nmap
  • shortport
  • stdnse
  • string
  • table
