脚本使用类型: portrule

脚本下载地址: http://nmap.org/svn/scripts/snmp-sysdescr.nse

User Summary

Attempts to extract system information from an SNMP version 1 service.

Script Arguments

snmpcommunity

See the documentation for the snmp library.

Example Usage

执行格式 

nmap -sU -p 161 --script snmp-sysdescr <target>
