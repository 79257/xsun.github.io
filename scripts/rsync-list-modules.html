File rsync-list-modules

脚本使用类型: portrule
脚本所属分类: discovery, safe
脚本下载地址: http://nmap.org/svn/scripts/rsync-list-modules.nse

User Summary

Lists modules available for rsync (remote file sync) synchronization.

Example Usage

执行格式 

nmap -p 873 --script rsync-list-modules <ip>

Script Output 

PORT    STATE SERVICE
873/tcp open  rsync
| rsync-list-modules: 
|   www            	www directory
|   log            	log directory
|_  etc            	etc directory

Requires

  • rsync
  • shortport
  • stdnse
